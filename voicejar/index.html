How it works

按住 Fn 说话

浮窗自动出现在光标附近，实时显示识别文字。

松开 Fn

文字立即输入到光标位置，零延迟上屏。

双击 Fn 切换模式

红点 = 即时模式，紫点 = 润色模式。AI 帮你整理后再上屏。

Getting Started

  1. 下载 .dmg 并打开，将 VoiceJar 拖入 Applications
  2. 首次打开右键选择「打开」
  3. 按引导授权辅助功能、麦克风、语音识别
  4. 按住 Fn，开始说话
